# 每日时间块消消乐 — 规格文档

## Concept & Vision

每天的时间安排变成一条垂直时间轴，每个时间块是一个可交互的方块。点击"完成"后，方块以弹射动画消失，配合得分反馈，让打卡变得有成就感和游戏感。整体氛围：简洁、轻松、有一点趣味。

## Design Language

- **美学方向**：轻量游戏感 + 极简时间管理工具。柔和的背景色，圆角卡片，清脆的点击反馈。
- **配色**：
  - 背景：`#F7F6F3`（暖白）
  - 时间轴线：`#D4D0C8`
  - 待办卡片：白色 `#FFFFFF`，圆角 12px，subtle shadow
  - 进行中（hover/press）：`#FFF8E7`（暖黄高亮）
  - 已完成：`<-- 消除动画，无颜色 -->`
  - 时间标签：`#999`，字体等宽
  - 得分/金币：`#F5A623`（金黄色）
- **字体**：
  - 标题/时间：system-ui 或 -apple-system（iOS 风格）
  - 主题色：`#2D2D2D`
- **动效**：
  - 点击完成：方块缩小 + 向上飘出 + 淡出（300ms ease-out）
  - 完成时：轻微的震动反馈（CSS transform shake）
  - 时间轴连接线随方块消除逐渐变短
- **图标**：无外部依赖，用纯 CSS 绘制简单图形

- 垂直时间轴，左侧时间标签，右侧卡片方块
- 时间轴用虚线连接各时间点

## Features & Interactions

### 核心功能

1. **时间块展示**：根据预设的时间表，从 localStorage 读取今日数据，渲染所有时间块
2. **点击完成**：点击方块 → 弹射动画 → 消除 → 金币+1 → localStorage 同步
3. **数据持久化**：完成状态存 localStorage，key=`dayplan_YYYY-MM-DD`
4. **次日自动重置**：每次打开时检测日期，若日期变更则重置所有方块

### 交互细节

- **点击方块**：按下时方块缩小至 0.95，放开后触发消除动画
- **消除动画**：translateY(-20px) + scale(0.8) + opacity(0)，300ms
- **金币动画**：+1 飘字效果，上浮消失
- **已完成数**：顶部显示 "完成 X/Y"
- **空状态**：所有方块消除后显示 "🎉 今日全部完成！"

### 数据结构

```javascript
// localStorage key: "dayplan_2026-05-30"
{
  date: "2026-05-30",
  blocks: [
    { id: 1, time: "06:00", label: "游泳",  done: false, doneAt: null },
    { id: 2, time: "08:30", label: "聚焦工作", done: true,  doneAt: "2026-05-30T09:15:00" },
    ...
  ],
  coinCount: 3  // 历史累计金币
}
```

## Component Inventory

### 顶部栏
- 左侧：日历图标 + 今日日期
- 右侧：金币计数器（带金币 emoji）

### 时间轴线
- 左侧垂直虚线，圆点（●）标记每个时间节点
- 时间节点用小圆点（8px）连接

### 时间块卡片
- 白色圆角卡片，右边时间轴线引出
- 左上：时间段（灰色小字）
- 中间：事项名称
- 右下：若无内容则显示时间
- 点击：亮黄色高亮 → 消除动画

### 完成飘字
- "+1 🪙" 从点击位置向上飘出，1秒后消失

## Technical Approach

- **单文件 HTML**：所有 CSS/JS 内联，无需构建
- **数据存储**：localStorage，JSON 序列化
- **无外部依赖**：纯 HTML/CSS/JS，不引入任何库
- **移动端优先**：max-width: 480px 居中，全屏高度布局
- **预设时间表**：hardcoded 在 JS 里，或从 URL 参数初始化
